## 安装

*以Mac OS X 11.0.1为例*

通过 `homebrew` 安装

~~~sh
brew install ansible
~~~

安装完成后手动创建部分需要的配置文件

~~~sh
# 创建ansible工作目录
sudo mkdir /etc/ansible && cd /etc/ansible

# 创建默认Inventory
vi hosts
echo 192.168.10.1 >> hosts

# 创建配置文件
touch ansible.cfg
~~~
